## 📝 自定义直播源

编辑 `assets/config/channels.json` 添加自己的源：

```json
{
  "channels": [
    {
      "id": "my_channel",
      "name": "我的频道",
      "url": "http://your-stream-url.m3u8",
      "group": "自定义"
    }
  ]
}
```

### 支持的格式

- ✅ `.m3u8` (HLS) - 推荐
- ✅ `.mp4` (点播)
- ⚠️ `.flv` (部分支持)
- ❌ RTMP (不支持)

## 🔍 故障排查

### 视频无法播放

1. 检查网络连接
2. 尝试其他频道
3. 更新直播源配置
4. 重启应用

### 卡顿/缓冲

1. 降低网络负载 (关闭其他应用)
2. 切换 WiFi/有线网络
3. 联系直播源提供商
